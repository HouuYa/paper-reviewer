Gemini API와 PDF 분석 앱

이 Streamlit 앱은 Google의 Gemini API를 사용하여 PDF 문서를 분석하고 질문에 답변하는 도구입니다.

기능:
- PDF 파일 업로드
- Gemini API를 사용한 PDF 내용 분석
- 사용자 질문에 대한 AI 기반 답변 생성

설치 방법:
1. 이 저장소를 클론합니다:
   git clone [저장소 URL]

2. 프로젝트 디렉토리로 이동합니다:
   cd [프로젝트 디렉토리]

3. 필요한 라이브러리를 설치합니다:
   pip install streamlit google-generativeai PyPDF2

사용 방법:
1. 터미널에서 다음 명령어를 실행하여 앱을 시작합니다:
   streamlit run gemini2.py

2. 웹 브라우저에서 앱이 열립니다.

3. PDF 파일을 업로드합니다.

4. Gemini API 키를 입력합니다.

5. PDF 내용에 대한 질문을 입력합니다.

6. AI가 생성한 답변을 확인합니다.

주의사항:
- Gemini API 키는 안전하게 관리해야 합니다. 공개 저장소에 API 키를 올리지 마세요.
- 대용량 PDF 파일의 경우 처리 시간이 길어질 수 있습니다.
